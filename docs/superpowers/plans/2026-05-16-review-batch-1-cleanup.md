# Review Batch 1 Cleanup

**Date:** 2026-05-16

**Scope:** Release Smoke And QA plus Cleanup And Generated Artifacts.

## Decision

Generated browser evidence is useful during smoke testing, but it should not appear as source work during release review. The repo now ignores:

- `/output/`
- `/.playwright-cli/`

These folders may still exist locally because they contain screenshots, export files, and console snapshots from recent smoke work. They are not source code and should be regenerated by the smoke scripts when needed.

## What Stays Reviewable

Release smoke source remains visible and reviewable:

- `package.json`
- `scripts/release-smoke-matrix.test.ts`
- `scripts/smoke-*.mjs`
- `scripts/smoke-*.ts`
- `scripts/smoke-*.tsx`
- `scripts/notification-queue-audit.ts`

## What Does Not Get Deleted Automatically

Do not blindly delete generated evidence. Some screenshots are useful when reviewing UI details with the in-app browser. Cleanup means keeping review surfaces clear, not erasing proof.

## Proof

Required checks for this cleanup:

- `git status --short` no longer lists `/output/` or `/.playwright-cli/` generated files.
- `git diff --check`
- `npm test -- scripts/release-smoke-matrix.test.ts`
- `npm run smoke:release`

## Result

Completed on 2026-05-16. Generated browser and smoke evidence is ignored for source review, the release smoke matrix test passed, and `npm run smoke:release` passed end to end after the cleanup.
